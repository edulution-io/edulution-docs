## Diagnose-Befehle

Verwenden Sie diese Befehle, um Verbindungsprobleme zu diagnostizieren:

### IMAP-Verbindung testen

<CodeBlock
  title="IMAP Port 993 testen"
  description="Überprüfen Sie, ob der IMAP-Server erreichbar ist"
  variants={[
    {
      platform: 'Windows',
      command: 'Test-NetConnection -ComputerName {{mailDomain}} -Port 993',
      expectedOutput: 'TcpTestSucceeded : True'
    },
    {
      platform: 'Linux',
      command: 'telnet {{mailDomain}} 993',
      expectedOutput: 'Trying 123.456.789.0...\nConnected to {{mailDomain}}.\nEscape character is \'^]\'.'
    },
    {
      platform: 'macOS',
      command: 'nc -zv {{mailDomain}} 993',
      expectedOutput: 'Connection to {{mailDomain}} port 993 [tcp/imaps] succeeded!'
    }
  ]}
/>

### SMTP-Verbindung testen

<CodeBlock
  title="SMTP Port 465 testen"
  description="Überprüfen Sie, ob der SMTP-Server erreichbar ist"
  variants={[
    {
      platform: 'Windows',
      command: 'Test-NetConnection -ComputerName {{mailDomain}} -Port 465',
      expectedOutput: 'TcpTestSucceeded : True'
    },
    {
      platform: 'Linux',
      command: 'telnet {{mailDomain}} 465',
      expectedOutput: 'Trying 123.456.789.0...\nConnected to {{mailDomain}}.\nEscape character is \'^]\'.'

      platform: 'macOS',

    }
  ]}
/>

### SSL-Zertifikat überprüfen

<CodeBlock
  title="Zertifikat prüfen"
  description="Zeigt Details des SSL-Zertifikats an"
  variants={[
    {
      platform: 'Windows',
      command: '$cert = Invoke-WebRequest -Uri https://{{mailDomain}} | Select-Object -ExpandProperty BaseResponse | Select-Object -ExpandProperty ServerCertificate\n$cert | Format-List Subject, Issuer, NotAfter',
      expectedOutput: 'Subject : CN={{mailDomain}}\nIssuer  : CN=Let\'s Encrypt Authority X3\nNotAfter: [Ablaufdatum]'
    },
    {
      platform: 'Linux',
      command: 'openssl s_client -connect {{mailDomain}}:993 -showcerts 2>/dev/null | openssl x509 -noout -subject -issuer -dates',
      expectedOutput: 'subject=CN = {{mailDomain}}\nissuer=C = US, O = Let\'s Encrypt, CN = R3\nnotBefore=[Datum]\nnotAfter=[Datum]'
    },
    {
      platform: 'macOS',
      command: 'openssl s_client -connect {{mailDomain}}:993 -showcerts 2>/dev/null | openssl x509 -noout -subject -issuer -dates',
      expectedOutput: 'subject=CN = {{mailDomain}}\nissuer=C = US, O = Let\'s Encrypt, CN = R3\nnotBefore=[Datum]\nnotAfter=[Datum]'
    }
  ]}
/>

### Netzwerk-Latenz testen

<CodeBlock
  title="Ping Test"
  description="Misst die Verbindungsgeschwindigkeit zum Server"
  variants={[
    {
      platform: 'Windows',
      command: 'ping {{mailDomain}} -n 4',
      expectedOutput: 'Antwort von [IP]: Bytes=32 Zeit<50ms TTL=64'
    },
    {
      platform: 'Linux',
      command: 'ping -c 4 {{mailDomain}}',
      expectedOutput: '64 bytes from {{mailDomain}}: icmp_seq=1 ttl=64 time=25.3 ms'
    },
    {
      platform: 'macOS',
      command: 'ping -c 4 {{mailDomain}}',
      expectedOutput: '64 bytes from {{mailDomain}}: icmp_seq=1 ttl=64 time=25.3 ms'
    }
  ]}
/>
